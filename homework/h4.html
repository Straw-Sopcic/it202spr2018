Questions:

  1. What is the Box Model?
    2. Every html element is contained within its own box whose properties can be adjusted.
  2. What does it mean to be a "block-level" element?
    3. Elements that take up usually the full width of their parent structure and are surrounded top an bottom by new lines.
  3. Provide one tag that is, by default, rendered as a block-level element.
    4. "p" tag
  4. What is an "inline" element?
    5. An element that attempts to appear directly next two its parent element or the element it is associated with
  5. Provide one tag that is, by default, rendered as an inline element.
    6. "label" tag
  6. What are the two ways we should include CSS in our pages?
    7. Using "style" element for mods within the same html file.
    Using "link" element for mods within external file
  7. What is the function of a selector in a CSS rule?
    8. It gives us more accurate and effiecient control over styling of either multiple objects of the same type or a single and very specific object.
  8. Write a CSS rule to set the background color of the body element to #0000FF
